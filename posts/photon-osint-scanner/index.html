🔍 Mastering Photon: The Ultimate Web Scraping & OSINT Tool for Ethical Hackers

July 3, 2025 · Rajkumar Kumawat

🌐 What is Photon?#

Photon is a lightning-fast, smart web crawler built specifically for OSINT tasks. It’s not just a URL scraper—it’s a powerful weapon for bug bounty hunters, ethical hackers, and red teamers to extract:

  • ✅ Emails & social links
  • ✅ Endpoints & APIs
  • ✅ JS files & secrets
  • ✅ Hidden files (PDFs, ZIPs)
  • ✅ Wayback Machine archives

⚠️ Always scan responsibly — only target websites you own or have permission to test.

🚀 Why Use Photon?#

Feature Benefit
🧠 Intelligent Crawling Finds links, parameters, scripts & secrets with minimal noise
🔍 JS Parsing Uncovers hidden API endpoints from obfuscated JavaScript
⌛ Wayback Archive Support Digs up old/deleted pages from Wayback Machine
🎯 Regex Intelligence Auto-detects emails, AWS keys, and more using predefined regex patterns
💾 Output Control Supports .json, .txt, and .csv for easy parsing in automation

🛠️ How to Install Photon (Kali Linux / Ubuntu)#

Option 1: Clone from GitHub#

git clone https://github.com/s0md3v/Photon.git
cd Photon
pip3 install -r requirements.txt

Option 2: Use pip#

pip3 install photon

📌 Basic Usage Syntax#

python3 photon.py -u https://target.com -o output_folder
  • -u : Target URL
  • -o : Folder to store output

⚙️ Photon Command Cheat Sheet#

Command Purpose
-u Target URL
-o Output directory
--emails Extract emails only
--wayback Include Wayback Machine results
--js Scan JS files for endpoints
--only-urls Only grab URL links
--threads 20 Speed up scan with multithreading
--delay 2 Delay requests to avoid detection

📁 Output Files Explained#

Photon gives you detailed categorized results. Here’s a breakdown of what you’ll see:

File Name Description
endpoints.txt API endpoints and parameterized URLs
external.txt External domains linked from the site
files.txt Downloadable files like PDF, DOCX, ZIP
fuzzable.txt URLs with parameters — good for fuzzing
intel.txt Leaked secrets, emails, keys
internal.txt All internal links found
robots.txt Crawled site’s robots.txt
scripts.txt JavaScript files
report.json Combined report for automation tools

📌 Use grep, ffuf, or custom scripts on these files for deeper analysis.

🧪 Example: Scanning Accenture.com#

python3 photon.py -u https://www.accenture.com -o sherlock

Sample Output (simplified):#

www.accenture.com/
├── endpoints.txt         ➜ 6249 endpoints
├── external.txt          ➜ External links (Google, LinkedIn, etc.)
├── files.txt             ➜ Found: .pdf, .ppt, .docx
├── fuzzable.txt          ➜ 9000+ URLs with parameters
├── intel.txt             ➜ Emails + API keys (if any)
├── internal.txt          ➜ Full link map of the domain
├── robots.txt            ➜ Disallowed paths
├── scripts.txt           ➜ JavaScript endpoint sources

🔍 Real-World OSINT Examples#

🕵️ Find Emails#

photon -u https://target.com --emails

➡ Outputs:

contact@target.com  
admin@target.com

🕸️ Discover API Endpoints#

photon -u https://target.com --js

➡ Extracts:

/v1/user/profile  
/v1/admin/dashboard

🕰️ Wayback Recon#

photon -u https://oldsite.com --wayback

➡ Results:

/old-login.html  
/dev-test.php

🧱 Defensive Tips for Admins#

If you’re on the blue team:

Tip Purpose
Block bots via robots.txt Prevent crawlers
Rate-limit requests Detect automated scans
Obfuscate JS code Hide internal logic
Monitor unusual User-Agents Photon uses Python headers

🧪 Practice Lab (Safe & Legal)#

1. Run Photon on your own blog:
   photon -u https://yourblog.com -o output

2. Open `internal.txt` to find hidden pages

3. Use `intel.txt` to see if secrets/emails leak

4. Load `scripts.txt` into LinkFinder for JS analysis
